BOSQICH 11 —  MULTI-TENANT REDESIGN (v3.1)
================================================================================
Maqsad: /pair'siz UX, har user uchun izolyatsiyalangan guruh ro'yxati,
        token-based App↔Bot ulanishi, ChatAction-driven group registry.

11.1  Storage:
        - active_groups → active_groups:<owner_id> (per-owner hash)
        - pair_code* → connect_token:<token> {user_id, created_at}
        - enrolled_users (set) — /start yuborganlar audit
        - remove_group_from_all_owners(group_id) — ChatAction tozalash

11.2  Bot handlerlar:
        - pairing.py o'chirildi
        - enrollment.py qo'shildi:
          • /start [token] — token bo'lsa: bind_connect_token; aks holda welcome
          • /here (guruhda) — add_active_group(sender_id, ...)
          • /unhere (guruhda) — remove_active_group(sender_id, chat_id)
          • /groups — shu user'ning per-owner ro'yxati
          • ChatAction listener — bot kicked → remove_group_from_all_owners

11.3  HTTP API:
        - /pair, /auto-pair olib tashlandi
        - POST /connect-init {token} → {t_me_url}
        - GET /connect-status?token → {ready, user_id}
        - POST /connect-claim {token} → {credentials, user_id, groups}
        - GET /groups?user_id (HMAC) — per-owner

11.4  App:
        - LoginFrame o'chirildi
        - connect_flow.py: token gen → init → webbrowser.open(t.me/?start=) → poll → claim
        - gui/connect_dialog.py: modal + countdown
        - window.py: bootstrap (cred bor → Main; yo'q → connect)

11.5  MainFrame:
        - cred ichidagi static groups o'rniga GET /groups dinamik
        - 30s auto-refresh (Tk after)
        - "🔄 Yangilash" tugmasi (manual refresh)
        - logout → bootstrap (yangi connect)

11.6  Multi-tenant invariantlar:
        - A va B bir guruhga /here → har biri o'z ro'yxatiga qo'shadi
        - GET /groups?user_id=A faqat A'niki, B'niki ko'rinmaydi
        - app_status (user_id, chat_id) — alohida saqlanadi
        - stop_signal (user_id) — alohida

11.7  Testlar:
        - test_storage_json.py: per-owner isolation, connect_token CRUD
        - test_api_routes.py: connect oqimi, /groups HMAC + per-owner
        - test_smoke.py: import yangi modullar
        - 97 test yashil

DoD:
